# 🎵 Audio Reactive Waves

### 🔴 [Click Here for Live Demo](https://themurad.github.io/Audio-Reactive-Waves/)

### Installation

1.  **Clone the repository**
    ```bash
    git clone [https://github.com/TheMurad/Audio-Reactive-Waves.git](https://github.com/TheMurad/Audio-Reactive-Waves.git)
    ```

2.  **Navigate to the project directory**
    ```bash
    cd Audio-Reactive-Waves
    ```

3.  **Run the project**
    * Since this project uses vanilla HTML/JS, you can simply open `index.html` in your browser.
